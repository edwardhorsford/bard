# Variables

```ruby
irb(main)> x = 15
irb(main)> y = 20
irb(main)> x + y
=> 35
```

Variables are used to store values. They can be used to store a value of any
data type.

## Data Types
* Number
* Float
* String
* Booleans

## Naming Conventions:
- Camel Case `variableName`
- Snake Case: `variable_name`git push
